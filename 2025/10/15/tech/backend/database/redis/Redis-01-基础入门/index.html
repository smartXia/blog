Redis基础入门

什么是Redis

Redis是一个高性能的键值对数据库，数据存储在内存中，读写速度很快。

主要特点：

  • 内存存储，速度快
  • 支持数据持久化
  • 支持多种数据类型
  • 单线程模型
  • 每秒可处理10万次读写操作

Redis应用场景

  1. 缓存系统 - 热点数据缓存，减少数据库压力
  2. 会话存储 - 分布式Session，用户登录状态管理
  3. 消息队列 - 异步任务处理，发布/订阅模式
  4. 分布式锁 - 防止重复操作，保证数据一致性
  5. 计数器 - 网站访问统计，实时数据统计

Redis优缺点

优点：

  • 读写速度极快
  • 支持多种数据类型
  • 支持数据持久化
  • 支持主从复制
  • 所有操作都是原子性的

缺点：

  • 数据量受内存限制
  • CPU密集型任务性能有限
  • 主从同步可能有延迟
  • 在线扩容较复杂
